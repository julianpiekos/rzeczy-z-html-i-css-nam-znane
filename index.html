dfn co zaraz będzie definiowane w tym samym tagu co tag dfn
dl lista definicji, w nim tagi: dt, dd

fieldset objekt z czarną solidną ramką
form w nim tagi formularza np. input i select
h1-h6 coraz mniejsze nagłówki(h1 to największy)

Atrybuty
Atrybut Opis
action do tagu form, ścieżka pliku (zwykle PHP) do którego mają być presłane dane z tego formularza po jego przesłaniu
alt do tagu img, tekst który się wyświetli jeżeli obraz się nie załaduję
autocomplate do tagu input typu text, wartości on i off, czy przeglądarka może automatycznie uzupełnić dane pole tekstowe
autofocus do tagu input typu text, atrybut pusty, jeżeli jest to po włączeniu strony automatycznie jest włączone wpisywanie tekstu do tego pola
charset do tagu meta, ustawia kodowanie tekstu (np. UTF-8) dla tego dokumentu
checked do tagu input typu radio lub checkbox, atrybut pusty, jeżeli jest to pole wyboru jest automatycznie zaznaczone po załadowaniu strony
class dla wszystkich tagów, klasa obiektu, w css-ie lub w js z pomocą querySelectorAll można się odwołać przy użyciu .nazwa-klasy
colspan do tagu td, rozciągnięcie komórki w prawo
cols do obiektu textarea, szerokość obiektu textarea w znakach
content do tagu meta po atrybucie name, wartość danego metadanego
data-blablabla dla wszystkich tagów, zamiast 'blablabla' może być cokolwiek innego, atrybut tylko przechowujący tekst i nic nie robiący
for do tagu label, dla tagu z jakim id ma być ta etykieta (trzeba wpisać np. grupa-wiek-2)
height wysokość elementu (ustawia cechę CSS o takiej samej nazwie)
href do tagów które się odnoszą do jakiejś strony np. 'a' lub 'link', adres URL lub ścieżka pliku
id dla wszystkich tagów, identyfikator obiektu, w css-ie lub w js z pomocą querySelector można się odwołać przy użyciu #nazwa-identyfikatora
lang dla wszystkich tagów, język tekstu w obiekcie, zwykle używany w tagu html
maxlength do tagu input typu text, ile maksymalnie znaķów można wpisać w polu tekstowym
method do tagu form, ustawia metodę przesyłania danych z formularza (np. POST lub GET)
name w tagu input w inpucie typu text oznacza nazwę wartości po przesłaniu formularza, w inpucie typu radio lub checkbox oznacza nazwę pytania i wtedy stosuje się ją do kilku inputów
name w tagu meta nazwa metadanego którego wartość jest potem określona w tym samym elemencie
placeholder do tagu input typu np. text, tekst który pokazuje się w polu tekstowym na szaro zanim się coś wpisze
rel do tagu link, relacja między plikami html i ty z którym się łączy (np. stylesheet)
required do tagu input typu np. text lub date, pole wymagane
rowspan do tagu td, rozciągnięcie komórki w dół
rows do tagu textarea, wysokość obiektu textarea w wierszach
size do tagu input, szerokość pola tekstowego
src do tagów które odnoszą się do plików (nie z kodem), adres URL lub ścieżka pliku nie z kodem
style do wszystkich tagów, style CSS dla danego obiektu
svg jakimś sposobem robi obrazek z tagiem pach z atrybutem d w środku i kilkoma atrybutami (kody do ikonek svg na różnych stronach)
target do tagu a, jak otworzyć link (np. _blank aby otworzyć w nowej karcie)
title do wszystkich tagów, tekst który się wyświetla w malutkim białym z czarną ramķą okienku po najechaniu
type do tagów input i tagów które się do czegoś odnoszą, w input=typ inputa, w tagach które się do czegoś odnoszą=format pliku do którego się odnosi (np. audio/mp3)
value do tagu input, w inpucie typu text=domyślna wartość, w inpucie typu checkbox lub radio=wartość pytania jezeli wybierzę się tą opcje
width szerokość elementu (ustawia cechę CSS o takiej samej nazwie)



Cechy CSS
Cecha Opis
align-items centruje w osi y wszystko z tego kontenera w ramach jego
background-attachment jak ma się zachowywać tło przy przewijaniu, wartości: fixed=tło się przesuwa przy scrollowaniu, scroll=tło się powtarza przy przewijaniu
background-color kolor tła obiektu/td>
background-image: url('') obraz tła, między apostrofami lokalizacja lub URL obrazka
background-size wielkość obrazu tła (zwykle ustawiane na 100% czyli na cały objekt)
border ramka, wartość to: [grubość ramki] [typ ramki np. solid lub dotted] [kolor ramki]
border-collapse czy ramki się łączą, wartości: collapse=łączą się (przydatne przy tabelach), i separated=nie łączą się i jest to domyślna wartość
border-[strona] tak samo jak border tylko dla jednej strony objektu
border-[strona]-style druga wartość z border tylko dla jednej strony
border-[strona]-width pierwsza wartość z border tylko dla jednej strony
border-[strona]-color trzecia wartość z border tylko dla jednej strony
border-radius promień zaokrąglenia bordera
border-style druga wartość z border
box-sizing czy width i height robi szerokość wysokość łączną: content, padding i border czy tylko content, wartości: border-box=łączna, content-box=tylko content
color kolor tekstu
content: "" używane tylko przy ::after lub ::before, między tymi cudzysłowami tekst który jest wyświetlany na początku lub na końcu tego tagu
cursor jaki kursor po najechaniu na objekt (np. pointer)
display typ wyswetlania, inline = rozmiar contentu to rozmiar tekstu, none = element nie jest wyświetlany, block = element domyślnie zajmuje cąłą dostępną szerokość i można ustawiać width i height, inline-block = domyśnie jak inline ale można zmieniać rozmiar, flex = ustawia wszystkow środku w jednej lini
font-family hierarhia czcionek lub grup czcionek (np. Courier New', Courier, monospace lub Arial, Helvetica, sans-serif), pierwsza a jak niedostępna to kolejna czcionka i tak dalej
font-size rozmiar czcionki
font-style typ wybranej czcionki, italic = kursywa, oblique = wymuszone przechylenie czcionki
font-variant small-caps = małe litery są konwertowane na wielkie ale mają mniejszy rozmiar niż normalne wielkie
font-weight pogrubienie czcionki, 100, 200, 300 itd. do 900 = jak bardzo pogrubiona czcionka, noramal = brak pogrubienia, bold = taki efekt jak przy tagu bold
height wysokość contentu lub contentu, paddingu i bordera
justify-content centruje w osi x wszystko z tego kontenera w ramach jego
letter-spacing odległość między literami
line-height wysokość lini (ale nie wysokość tekstu)
margin margines do okoła
margin-right margines do z prawej
opacity nieprzezroczystość obiektu w skali od 0 do 1 (1 to zupełnie nieprzezroczysty)
overflow jak ma się zachowywać tekst który wyjdzie z kontenera tagu w którym jest, hidden = ukrywa nadmierność tekstu, scroll = tworzy pasek do przesuwania w kontenerze tego tagu, aby dało się przesuwać i zobaczyć całość
padding wypełnienie do okoła elementu
padding-bottom wypełnienie na dole elementu
position wartości position
[strona] określanie odległości od jakiejś strony strony, dla cechy position
text-align wyruwnanie tekstu, center = na środku, end = na koniec szerokości, start = na początek szerokości
text-decoration dekoracja tekstu, np. '2px dotted red' robi ramkę z kropek o wielkości 2x2 koloru czerwonego
text-indent wcięcie tekstu od lewej
text-transform capitalize = pierwsza litera w zawartości tagu wielką literą
white-space jak odczytywać białe znaki, pre-line = odczytywane są wszystkie tabulatory, pre-wrap = odczytywane są wszystkie białe znaki, normal = domyślne
width szerokość contentu lub contentu, paddingu i bordera
word-spacing odległość między wyrazami
z-index nr warstwy, czyli jeżelki 1 objekt ma z-index = 2 a 2 objekt z-index = 1, to jeżeli na siebie nachodzą za pomocą position: absolute lub fixed ten 1 jest na górze ale bez obu z-index na górze byłby ostatni czyli 2



Selektory CSS
Coś z Selektora Opis
, w środku określania tagu/tagów, kolejny selektor do tej grupy i potem ustawia się cechy dla wszystkich selektorów z tej grupy
spacja w środku określania tagu/tagów, tag po spacji w tagu przed spacją i może być o dużo kontenerów oddalony ale i tak działa
> w środku określania tagu/tagów, tag po tym znaku bezpośrednio w tagu przed tym znakiem
# na początku określania tagu, id tagu który ma być wybrany po tym znaku
. na początku określania tagu/tagów, klasa tagu/tagów który/które ma/mają być wybrany/wybrane po tym znaku
nazwa-tagu. całe określanie tagu/tagów, dany typ tagu z daną klasą
* całe określanie tagów, wszystkie tagi
nazwa-tagu całe określanie tagu/tagów, wszystkie tagi tego typu
:hover na końcu selektora, po najechaniu na tag/tagi z tego selektora
:link na końcu selektora, tylko dla tagu a, nieodwiedzony link
:visited na końcu selektora, tylko dla tagu a, odwiedzony link
:active na końcu selektora, tylko dla tagu a, aktualnie klikany link
::before na końcu selektora, tworzy tag z danymi cechami i takią zawartośćą jak między apostrofami w content: '' przed tym/tymi tagiem/tagami
::after na końcu selektora, tworzy tag z danymi cechami i takią zawartośćą jak między apostrofami w content: '' po tym/tych tagu/tagach
::first-letter na końcu selektora, wybiera tylko pierwszą literę zawartości tego tagu
